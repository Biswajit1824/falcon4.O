## Arithmatic operator :
  - "+" : Addition (ex - 2+5 = 8)
  - "-" : Subtraction (ex - 5-2 = 3)
  - "*" : Multiplication (ex - 5*2 = 10)
  - "/" : Division (ex - 10/2 = 5)
  - "%" : Modulus (ex - 5%2 =1)
  - "**" : Exponent (ex - 2**3 = 8)
